# 💳 Personal Expense Tracker

A professional fintech-style Personal Expense Tracker built using **Python, Streamlit, SQLite, Pandas, and Plotly**.

This application allows users to:

- 🔐 Create accounts and login securely
- 💰 Track personal expenses
- 📊 Analyze spending habits
- 📈 Visualize expense trends
- 💸 Monitor monthly budgets
- 🗃 Store transaction history in SQLite database

---

# 🚀 Features

✅ User Authentication (Login / Signup)

✅ Budget Monitoring

# ⚙️ Installation Guide

## 1️⃣ Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/personal-expense-tracker.git
```

---

## 2️⃣ Open Project Folder

```bash
cd personal-expense-tracker
```

---

## 3️⃣ Install Requirements

```bash
pip install -r requirements.txt
```

---

## 4️⃣ Run Application

```bash
streamlit run app.py
```

# 💡 How It Works

1. User logs into the application  
2. User adds expenses manually  
3. Data gets stored in SQLite database  
4. Dashboard processes expense data  
5. Charts and KPIs are generated dynamically  
6. Budget insights are displayed in real-time
